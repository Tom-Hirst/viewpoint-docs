Dependant on your Viewpoint configuration, Users can create their own accounts by registering, however you may from time to time need to manually add users.

## Adding a new User

![User list](/assets/content/userlist.png)

1. Go to the Admin Menu
2. Click Settings > Users > `Add Users`

![Add User page](/assets/content/adduser.png)

3. Enter the users details, in the left hand column.
4. Select the User level from the dropdown field on the right - See [Permissions](permissions) for more information on User Permissions 
5. Tick the 'Send Welcome email' checkbox - This will send the user a welcome email that they can use to activate their account.
6  Click 'Save User'
